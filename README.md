# Library Management System

## Assignment Overview

This project is a Java-based Library Management System developed to demonstrate Object-Oriented Programming (OOP) concepts, Java Collections Framework, Exception Handling, and Java 8 features such as Streams and Lambda Expressions.

## Technologies Used

* Java 8+
* Collections Framework (HashMap, ArrayList)
* Exception Handling
* Java Streams API
* Lambda Expressions
* Git & GitHub

## Features Implemented

### Book Management

* Add books
* Remove books
* View all books

### User Management

* Register users
* Maintain user records

### Library Operations

* Issue books
* Return books
* Check book availability

### Search Functionality

* Search books by title using Java Streams

### Exception Handling

* Custom exception classes
* Validation of invalid operations

## How to Run

1. Clone the repository.
2. Open the project in VS Code, Eclipse, or IntelliJ IDEA.
3. Ensure Java 8 or higher is installed.
4. Run `Main.java`.
